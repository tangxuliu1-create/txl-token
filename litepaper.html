1. Introduction

Tang Xuliu (TXL) is an experimental ERC-20 token deployed on the Ethereum blockchain. The project is designed for personal exploration of decentralized infrastructure, on-chain asset mechanics, and token economics.

TXL is not intended as an investment product, security, or financial instrument. The purpose of this project is technical experimentation and transparent on-chain disclosure.

2. Token Overview

  • Token Name: Tang Xuliu
  • Symbol: TXL
  • Network: Ethereum Mainnet
  • Total Supply: 10,000,000 TXL
  • Supply Type: Fixed

3. Distribution & Locking

Initial Circulating Supply:

  • 1,000,000 TXL initially available
  • 9,000,000 TXL locked via on-chain vesting contracts

Locking Mechanism:

  • 9,000,000 TXL locked using decentralized smart contracts
  • Monthly release: 150,000 TXL
  • Liquidity pool (LP) tokens locked for 24 months
  • All locks are publicly verifiable on Ethereum mainnet

4. Use Case

TXL is used for testing liquidity mechanics, smart contract interactions, vesting structures, and general on-chain behavior.

The token does not represent ownership, governance rights, dividends, or profit-sharing.

5. Transparency

All token contracts, liquidity pools, and lockups are deployed on-chain and can be independently verified through Ethereum block explorers.

6. Disclaimer

TXL is a personal experimental project. It is not an investment product and carries inherent market risks. Users interact with the token at their own discretion.
