1. Introduction

Tang Xuliu (TXL) is an ERC-20 token deployed on the Ethereum blockchain, supporting on-chain interactions, liquidity-related activities, vesting mechanisms, and ecosystem participation within Ethereum.

TXL does not represent equity, ownership, or governance rights in any entity, and no guarantees of returns are made. All token-related activities and data are publicly verifiable on-chain.

2. Token Overview

  • Symbol: TXL

  • Supply Type: Fixed

3. Distribution & Locking

Initial Circulating Supply:

  • 1,000,000 TXL initially available
  • 9,000,000 TXL locked via on-chain vesting contracts

Locking Mechanism:

  • 9,000,000 TXL locked using decentralized smart contracts
  • Monthly release: 150,000 TXL
  • Liquidity pool (LP) tokens locked for 24 months
  • All locks are publicly verifiable on Ethereum mainnet

4. Transparency

All token contracts, liquidity pools, and lockups are deployed on-chain and can be independently verified through Ethereum block explorers.
