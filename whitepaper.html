1. Introduction

Tang Xuliu (TXL) is an independently developed ERC-20 token deployed on the Ethereum blockchain, focused on transparent token mechanics, long-term vesting research, and on-chain infrastructure experimentation.

TXL is published as a long-term public on-chain project with transparent disclosures and verifiable infrastructure references.

TXL does not represent equity, ownership, or governance rights in any entity, and no guarantees of returns are made. All token-related activities and data are publicly verifiable on-chain.

2. Token Overview

  • Token Name: Tang Xuliu
  • Symbol: TXL
  • Network: Ethereum Mainnet
  • Total Supply: 10,000,000 TXL
  • Supply Type: Fixed

3. Distribution & Locking

Initial Circulating Supply:

  • 1,000,000 TXL initially available
  • 9,000,000 TXL locked via on-chain vesting contracts

Locking Mechanism:

  • 9,000,000 TXL locked using decentralized smart contracts
  • Monthly release: 150,000 TXL
  • Liquidity pool (LP) tokens held in a multi-signature Safe wallet (verifiable on-chain)
  • All locks are publicly verifiable on Ethereum mainnet

The initial circulating supply is allocated for liquidity provisioning and on-chain participation.
