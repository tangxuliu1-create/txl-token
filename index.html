Project Overview

Tang Xuliu (TXL) is an experimental ERC-20 token deployed on the Ethereum blockchain. This project is created for personal exploration of on-chain assets, token mechanics, and decentralized infrastructure.

Tokenomics & Vesting

Total Supply: 10,000,000 TXL (Fixed Supply)

Initial Circulating Supply:

  • 1,000,000 TXL initially available
  • 9,000,000 TXL locked via on-chain vesting contracts

Vesting & Lock Details:

  • 9,000,000 TXL locked using decentralized smart contracts
  • Monthly release: 150,000 TXL
  • Liquidity pool (LP) tokens locked for 24 months
  • All locks are verifiable on Ethereum mainnet

The initial circulating supply is used for liquidity provisioning, testing, and on-chain experimentation.
